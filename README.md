# Shield SDKs

Official SDKs for the Shield tamper-proof evidence API.

## Available SDKs

| Language | Package | Install |
|----------|---------|---------|
| JavaScript/TypeScript | [@getshield/js](https://npmjs.com/package/@getshield/js) | `npm install @getshield/js` |
| Python | [shield-python](https://pypi.org/project/shield-python/) | `pip install shield-python` |
| Java | shield-java | `implementation 'dev.shield:shield-java:0.1.0'` |

## Documentation

Full API documentation: https://getshield.dev/docs
